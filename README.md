# Applied DevOps

This repository contains resources and materials for learning and applying DevOps principles and practices. The content is divided into three phases:

## Phases Overview:

### Phase 1: Foundations & CI
- Understanding the DevOps Culture
- Core Principles
- Version Control Systems (GIT)
- Continuous Integration & Build Automation (Jenkins)

### Phase 2: Configuration & Containers
- Software Configuration Management (Ansible, Puppet)
- Containerization & Orchestration (Docker, Kubernetes)
- Continuous Delivery and Deployment

### Phase 3: Operations & Cloud
- Blue-Green Deployments
- Infrastructure Monitoring (Prometheus, Elasticsearch)
- Cloud Platforms (AWS, Azure, Google Cloud)
- DevOps Culture
- Continuous Improvement & Learning

Navigate to the respective folders to explore detailed content for each phase.
